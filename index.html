Featured Projects

Analysis of 3D DNA Folding on Gene Expression

Conducted during my project tutoré at Centre de Biologie Intégrative, this project involved analyzing ChIP-seq data to identify protein-binding sites and Hi-C data to map 3D DNA interactions in C. elegans. This work aimed to elucidate the impact of DNA folding on gene expression regulation.

View Report

Bulk RNA-seq Deconvolution Pipeline

Developed during my internship at INSERM DR Grand Ouest, this project focused on creating a pipeline for deconvolving bulk RNA-seq data to estimate cell-type proportions in patients with follicular lymphoma. The project involved literature review, method implementation, and constructing a reference atlas.

View Report

Italian Wines Classification

This project involved developing classification models to predict Italian wine grape varieties. It utilized R and KNIME for exploratory data analysis (EDA) and machine learning to achieve accurate predictions.

Bakery Sales Analysis

This project includes a Streamlit app designed to analyze bakery sales data. It provides valuable insights into daily, hourly, and weekly sales trends and highlights both the top-selling and least popular products.
